¡Bienvenido a la Aplicación Web Vulnerable!

Esta aplicación contiene vulnerabilidades intencionales para fines de aprendizaje y pruebas.

Explora las diferentes secciones para identificar y explotar las vulnerabilidades:

Inyección SQL

Prueba la funcionalidad de inicio de sesión y búsqueda para explotar vulnerabilidades de inyección SQL.
Inyección de Comandos

Utiliza las herramientas de red y lectura de archivos para explotar vulnerabilidades de inyección de comandos.
Validación de Entrada Incorrecta

Prueba los formularios de registro y comentarios para explotar vulnerabilidades de validación de entrada.
SSRF (Server-Side Request Forgery)

Utiliza la funcionalidad de verificación de sitios web para explotar vulnerabilidades SSRF.

ADVERTENCIA: Esta aplicación es solo para fines educativos y de pruebas. No la utilices en entornos de producción o para atacar sistemas sin autorización.
